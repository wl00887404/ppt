# Selectors
## CSS Dinner
## 怎麼選擇出奇數元素？ ```css div:nth-child(odd){ /*code here...*/ } ```
## 怎麼選擇出最後的元素？ ```css div:last-child{ /*code here...*/ } ```
## 怎麼做出間隔？
# 設定全部 取消最後 ```css div{ margin-right:0.5em; } div:last-child{ margin-right:0; } ```
# 直接把非最後選出來 ```css div:not(:last-child){ margin-right:0.5em; } ```
# FlexBox
## 垂直置中 ```css #centercenter{ display:flex; justify-content:center; align-items:center; } ```
## grow 、 shrink 、 basic
+ grow 當寬度不足，自動填滿 + shrink 當寬度超過，自動壓縮 + basic 原始大小
