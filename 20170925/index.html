# 基本元件介紹 1
## 標籤 Label #### 用途是顯示文字 ![](./component1.jpg)

標籤 Label

屬性 說明
文本 Text 設定顯示的文字
字型大小 FontSize 設定文字大小，預設為 14
文本顏色 TextColor 設定文字顏色
字型大小 TextAlignment 設定文字對齊方式
## 文本輸入框 TextBox #### 用途是讓使用者輸入文字 ![](./component2.jpg)

文本輸入框 TextBox

屬性 說明
文本 Text 設定預設值
提示 Hint 設定提示文字，當使用者輸入後，就會消失
允許多行 MultiLine 設定是否可以換行
僅限數字 NumbersOnly 設定是否只能輸入數字
## 按鈕 Button #### 應用程式與使用者互動的主要元件 ![](./component3.jpg)

按鈕 Button

屬性 說明
啟用 Enabled 設定是否可按
圖像 Image 設定顯示圖片按鈕
顯示交互效果 ShowFeedback 設定按下按鈕，按鈕是否會閃動
## 水平布局 HorizontalArrangement #### 用途是排版，裡面的元件會以水平方式排列 ![](./component4.jpg)

水平布局 HorizontalArrangement

屬性 說明
AlignHorizontal 設定水平對齊方式
AlignVertical 設定垂直對齊方式
Visible 設定是否顯示元件
## 垂直布局 VerticalArrangement #### 跟水平布局，裡面的元件會以垂直方式排列 ![](./component5.jpg)

其他常用屬性

屬性 說明
背景顏色 BackgroundColor 設定背景顏色
粗體 FontBold 設定文字粗體
斜體 FontItalic 設定文字斜體
字體 FontTypeface 設定文字字體
高度 Height 設定元件高度
寬度 Width 設定元件寬度
